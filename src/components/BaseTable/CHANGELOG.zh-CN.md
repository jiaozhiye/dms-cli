# 更新日志

#### 发布周期

- 修订版本号：随着项目的进展，会根据需求随时发布更新。
- 次版本号：带有新特性的向下兼容的版本。
- 主版本号：含有破坏性更新和新特性，不在发布周期内。
- 组件升级：会保证现有的接口参数不变，完全向下兼容。
